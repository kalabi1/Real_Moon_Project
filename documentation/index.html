Moon Real Estate Management System

"Moon Real Estate Management System is a customized business management software. This software will provide complete Real Estate business solutions.
Here, Admin can manage Apartment Project and Land Project with the flat and plot details and sales property to the customer by full cash or installment.
Customer can able to login to the system by owner credential and manage own dashboard. Here, the user can see his/her purchase history, payment history and etc.

For better performance, on the front-end, we developed this software by two different applications. (i.e. Admin site and Client site application), on the other hand, back-end developed by one application."

How to setup(Back-End)

We are developed back-end site by using Spring Boot 2.0

  • 1) Import/open project by Intellij Idea or others IDE
  • 2) Change the application.properties and set your own credential
  • 3) Change the DBConfig class and set your own database credential
  • 4) Install Oracle OJDBC driver that present in your source file ojdbc7.jar

    •                          mvn install:install-file -Dfile=ojdbc7.jar -DgroupId=com.oracle -DartifactId=ojdbc7 -Dversion=12.1.0 -Dpackaging=jar

  • 5) Import all Maven depencies
  • 5) Build, test, package jar/war and deploy

How to setup(Front-End)

  • 1) Install angular package for both Admin and Client Application 

    •                       terminal-moon$ npm i
